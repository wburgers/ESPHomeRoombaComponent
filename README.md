# Roomba Component for ESPHome
A custom component written from scratch to receive sensor data from and send commands to an iRobot Roomba (that has no WiFi connection out of the box)

## Hardware
The gerber and schema folders provide a schema and board layout I've designed.
I'm not a professional hardware engineer, but a software engineer, so it may not be perfect, but it works.

## Software Setup/Use
Use the example roomba.yaml
